## 1. 职业能力倾向测验 (Job Aptitude Test)

### 1.1 资料分析 (Data Analysis) - **核心拿分项**
> **目标**: 满分 / 23分钟内完成

*   [ ] **核心概念 (Concepts)**
    *   [ ] 增长率 (Growth Rate): $r = \frac{现-基}{基}$
    *   [ ] 增长量 (Growth Amount): $Amount = \frac{现}{1+r} \times r$
    *   [ ] 比重 (Proportion): $\frac{A}{B}$
    *   [ ] 平均数 (Average): $\frac{Total}{Count}$
    *   [ ] 倍数 (Multiple): $\frac{A}{B}$
    *   [ ] 隔年增长率: $R = r1 + r2 + r1 \times r2$
    *   [ ] 混合增长率: 十字交叉法 / 线段法
*   [ ] **速算技巧 (Techniques)**
    *   [ ] 截位直除法 (Direct Division): 分母截三位
    *   [ ] 分数比较 (Fraction Comparison): 差分法、化同法
    *   [ ] 特征数字法 (Special Numbers): $1/7 \approx 14.3\%$, $1/9 \approx 11.1\%$
    *   [ ] 首数法 (First Digit): 看选项首位

### 1.2 言语理解与表达 (Verbal Comprehension) - **提分重点**
> **目标**: 正确率 80%+

*   [ ] **片段阅读 (Reading)**
    *   [ ] 主旨概括: 找中心句 (首尾句、转折关系、递进关系)
    *   [ ] 意图推断: “言外之意” (问题 -> 找对策; 现象 -> 找原因/呼吁)
    *   [ ] 细节理解: 对应原文，排除无中生有/偷换概念
    *   [ ] 标题填入: 概括最全、最吸睛
*   [ ] **逻辑填空 (Cloze)**
    *   [ ] 语境分析: 解释关系、反对关系 (转折)、并列关系
    *   [ ] 词义辨析: 感情色彩 (褒贬)、语义轻重、搭配对象
    *   [ ] 高频成语积累 (Top 200)

### 1.3 判断推理 (Reasoning) - **逻辑思维**
> **目标**: 稳定得分

*   [ ] **图形推理 (Graphic)**
    *   [ ] 位置规律: 平移、旋转、翻转
    *   [ ] 样式规律: 遍历、加减同异、黑白运算
    *   [ ] 属性规律: 对称、曲直、封闭 (开闭)
    *   [ ] 数量规律: 点、线、面、素 (元素)、角
    *   [ ] 空间重构: 六面体 (相对面、相邻面)
*   [ ] **逻辑判断 (Logic)**
    *   [ ] 翻译推理: 肯前必肯后，否后必否前 (逆否等价)
    *   [ ] 分析推理: 排除法、最大信息法
    *   [ ] 加强/削弱: 找论点，搭桥/拆桥，另有他因
*   [ ] **定义判断 (Definition)**: 提取关键词 (主客体、目的、方式)
*   [ ] **类比推理 (Analogy)**: 语义关系、逻辑关系、语法关系

### 1.4 数量关系 (Quantitative) - **战略性放弃/保底**
> **目标**: 只做简单题

*   [ ] **必会题型 (Must Do)**
    *   [ ] 工程问题: 赋值法 ($Total = Efficiency \times Time$)
    *   [ ] 行程问题: 基础公式，追及相遇
    *   [ ] 容斥原理: 两集合/三集合公式
*   [ ] **选做题型 (Optional)**
    *   [ ] 几何问题
    *   [ ] 排列组合
    *   [ ] 概率问题

### 1.5 常识判断 (General Knowledge) - **碎片积累**
> **目标**: 不丢基本分

*   [ ] 时政 (Current Affairs): 近一年重大会议、广西区情
*   [ ] 法律 (Law): 宪法、民法典、行政法
*   [ ] 科技/文史/地理 (General): 依靠日常积累

## 2. 综合应用能力 A类 (Composite Class A)

> **目标**: 掌握公文格式，锻炼逻辑表达

### 2.1 核心思维 (Core Thinking)
*   [ ] **角色代入**: 站在公务员角度思考 (依法行政、服务群众)
*   [ ] **归纳概括**: 从材料中提取要点 (摘抄 + 提炼)
*   [ ] **提出对策**: 针对问题给建议 (制度、监管、宣传、投入)

### 2.2 常见题型 (Question Types)
*   [ ] **案例分析 (Case Analysis)**
    *   [ ] 归纳概括题: 概括主要问题/原因/做法
    *   [ ] 提出对策题: 针对XX问题提出解决措施
    *   [ ] 应急处理题: 现场突发状况处理 (轻重缓急)
*   [ ] **公文写作 (Official Writing)**
    *   [ ] 通知 (Notice): 会议通知、整改通知
    *   [ ] 倡议书 (Proposal): 呼吁文明行为
    *   [ ] 宣传稿 (Publicity): 宣传活动亮点
    *   [ ] 回复/公开信 (Reply): 回应群众诉求
