## 🗣️ Part 1: 高频成语 (High-Frequency Idioms)

### 1. 一针见血 (yī zhēn jiàn xiě)
*   **释义**: 比喻说话直截了当，切中要害。
*   **近义**: 切中要害、一语中的、入木三分。
*   **Go语言类比**: 就像用 `grep` 精准定位到导致 bug 的那一行代码，而不是盲目地在整个代码库中搜索。
*   **真题语境**: "在分析问题时，我们要**一针见血**地指出根本原因，而不是停留在表面现象上。"

### 2. 对症下药 (duì zhèng xià yào)
*   **释义**: 针对病症用药。比喻针对具体情况采取有效措施。
*   **近义**: 有的放矢、因地制宜。
*   **Go语言类比**: 根据不同的错误类型（`error`、`panic`、`deadlock`），采用对应的处理策略，而不是用统一的 `try-catch` 模式。
*   **真题语境**: "解决基层治理难题，必须**对症下药**，针对不同地区、不同问题的特点，制定差异化的解决方案。"

### 3. 相辅相成 (xiāng fǔ xiāng chéng)
*   **释义**: 两件事物互相配合，互相辅助，缺一不可。
*   **近义**: 相得益彰、珠联璧合。
*   **Go语言类比**: `goroutine` 和 `channel` 的关系，两者配合才能实现高效的并发编程，单独使用都无法发挥最大效能。
*   **真题语境**: "政策制定与执行**相辅相成**，只有两者有机结合，才能确保政策落地见效。"

### 4. 背道而驰 (bèi dào ér chí)
*   **释义**: 朝着相反的方向奔跑。比喻彼此的方向或目标完全相反。
*   **近义**: 南辕北辙、分道扬镳。
*   **Go语言类比**: 你的代码目标是提高性能，但却使用了大量同步锁（`sync.Mutex`），导致并发性能下降，与目标**背道而驰**。
*   **真题语境**: "如果政策执行与政策初衷**背道而驰**，不仅无法解决问题，反而会引发新的矛盾。"

### 5. 举一反三 (jǔ yī fǎn sān)
*   **释义**: 从一件事情类推而知道其他许多事情。比喻善于学习，能够由此及彼。
*   **近义**: 触类旁通、闻一知十。
*   **Go语言类比**: 学会了 `interface{}` 的使用后，能够**举一反三**地理解 `generics`（泛型）的设计理念，因为它们都涉及类型抽象。
*   **真题语境**: "在总结工作经验时，要善于**举一反三**，从个别案例中提炼出普遍规律，指导今后的工作。"

## 📝 Part 2: 综应 A 类对策题模板 (Measure Template)

### 主题：对策建议题（根据问题提建议）
**【适用场景】**: 适用于材料中给出问题，要求提出对策、建议、措施等题目。

**【万能模板】**
> "针对材料中反映的_______（概括核心问题，如：服务效率低下、管理手段滞后、群众满意度不高）等问题，建议从以下几个方面着手解决：
>
> **一、强化顶层设计，完善制度保障。** _______（具体措施1，如：建立健全相关法律法规，明确责任主体和权责边界）。通过_______（方式/手段，如：制定实施细则、建立考核机制），确保_______（预期效果，如：各项工作有章可循、有据可依）。
>
> **二、创新工作方式，提升服务效能。** _______（具体措施2，如：运用数字化手段，推进'一网通办'）。通过_______（方式/手段，如：整合数据资源、优化办事流程），实现_______（预期效果，如：让数据多跑路、群众少跑腿）。
>
> **三、加强队伍建设，夯实人才基础。** _______（具体措施3，如：开展业务培训，提升专业素养）。通过_______（方式/手段，如：建立激励机制、完善考核评价），激发_______（预期效果，如：工作人员的内生动力和创造力）。
>
> **四、畅通反馈渠道，形成工作闭环。** _______（具体措施4，如：建立群众满意度评价机制）。通过_______（方式/手段，如：定期收集意见、及时整改落实），确保_______（预期效果，如：问题及时发现、及时解决，不断提升服务质量）。
>
> 总之，解决_______（核心问题）是一项系统工程，需要_______（总结性表述，如：统筹兼顾、久久为功），才能真正取得实效。"

**【逻辑拆解 (Go struct)】**
```go
type MeasureTemplate struct {
    ProblemSummary string   // 问题概括 ("针对材料中反映的...问题")
    Measures       []Measure // 对策列表
    Conclusion     string   // 总结性表述
}

type Measure struct {
    Title      string // 对策标题 ("一、强化顶层设计...")
    Specific   string // 具体措施 ("建立健全相关法律法规...")
    Method     string // 方式手段 ("通过制定实施细则...")
    Expected   string // 预期效果 ("确保各项工作有章可循...")
}
```

**【朗读示例】**
"针对材料中反映的社区养老服务设施不足、服务人员专业水平不高、老年人使用智慧设备困难等问题，建议从以下几个方面着手解决：

**一、强化顶层设计，完善制度保障。** 制定《社区养老服务设施建设标准》，明确新建小区必须配套建设养老服务设施。通过建立考核机制、纳入年度目标考核，确保政策落地见效。

**二、创新工作方式，提升服务效能。** 建设'智慧养老'平台，整合医疗、家政、餐饮等服务资源。通过'一键呼叫'、'上门服务'等方式，实现老年人足不出户享受便捷服务。

**三、加强队伍建设，夯实人才基础。** 开展养老服务人员专业技能培训，提升护理、心理疏导等能力。通过建立职业发展通道、提高薪酬待遇，吸引更多专业人才投身养老事业。

**四、畅通反馈渠道，形成工作闭环。** 建立老年人满意度评价机制，定期收集意见建议。通过设立服务热线、定期回访等方式，确保问题及时发现、及时解决，不断提升服务质量。

总之，解决社区养老问题是一项系统工程，需要政府、社区、家庭多方协同，久久为功，才能真正让老年人老有所养、老有所乐。"

## 💡 Part 3: 逻辑填空语境提示词 (Context Clues)

### 解释关系（同义/近义）
**【标志词】**: 即、也就是说、换言之、换句话说、也就是说、意思是、指的是、就是、等于、相当于

**【Go语言类比】**: 就像 `type alias`，`type UserID = int` 和 `int` 是同一个东西，只是换个名字。

**【示例】**
- "政策执行不力，**即**落实不到位，导致群众满意度下降。"
- "基层治理现代化，**换言之**，就是要运用新技术、新手段提升治理效能。"

### 反对关系（反义/对比）
**【标志词】**: 不是...而是、并非...而是、相反、反之、反而、却、然而、但是、可是、不过、虽然...但是

**【Go语言类比】**: 就像 `if-else` 语句，两个分支是互斥的，选择了一个就不能选择另一个。

**【示例】**
- "解决这个问题，**不是**简单地增加人手，**而是**要优化工作流程，提高效率。"
- "虽然投入了大量资金，**但是**效果并不理想，**反而**引发了新的问题。"
