## 1. 📝 原始材料 (Raw Data)

**标题**：价格战打不出未来！破"内卷"才有新动能
**来源**：半月谈新媒体 / 评论员：杨有宗

中央经济工作会议在部署2026年经济工作时强调，"坚持改革攻坚，增强高质量发展动力活力。制定全国统一大市场建设条例，深入整治'内卷式'竞争。"竞争本是市场经济的常态，但搞"内卷式"竞争、无休止地打价格战，只会助长"劣币驱逐良币"。只有深入整治"内卷式"竞争，引导企业依靠技术创新、产业升级练好"内功"，才能真正夯实高质量发展之基。

当前，不少行业进入新旧动能转换与存量竞争阶段，焦虑情绪传导之下，低价、低质、低水平竞争并不鲜见。超百款车型加入降价行列"贴身肉搏"、光伏组件低至每瓦六毛……这种"杀敌一千、自损八百"的打法，严重扭曲市场机制，损害市场公平竞争秩序。

以汽车行业为例，中国汽车流通协会发布的数据显示，2025年上半年，汽车经销商亏损比例上升至52.6%，74.4%的汽车经销商有不同程度的价格倒挂。该协会表示，严重的价格倒挂，可能导致服务缩水、售后减配等问题，最终损害消费者权益。

更为严峻的是，一些新兴行业和领域也陷入"内卷式"竞争的泥潭，不仅导致资源浪费、效率低下等问题，更为重要的是压缩企业创新空间。发展新质生产力，最忌讳"一窝蜂"。以当下最为火热的人形机器人产业为例。近日，国家发展改革委政策研究室副主任李超在新闻发布会上表示，随着新兴资本加速入场，我国目前已有超过150家人形机器人企业，这个数量还在不断增加，其中半数以上为初创企业或"跨行"入局。这对鼓励创新来讲是一件好事，但也要着力防范重复度高的产品"扎堆"上市、研发空间被压缩等风险。

繁荣表象之下，是同质化竞争的隐忧。在"内卷式"竞争压力下，有的机器人企业被迫将资源投向外观包装与演示优化，而非伺服电机、柔性关节等核心环节的攻关。长此以往，削弱的是未来的创新力和发展潜能，产品很可能因缺乏核心竞争力而昙花一现。

从更宏观的层面看，"内卷式"竞争还会造成"堵点"，阻碍国内大循环畅通，阻碍市场供给和需求间的高水平动态平衡，进而影响经济发展全局。

针对企业反映强烈的地方招商引资乱象，必须看到，一些地方政府盲目追风口、铺摊子，导致重复建设和落后产能增加。有的地方名目繁多地搞变相补贴，扭曲市场机制，导致资源错配。还有的地方，"保护主义"壁垒在一定程度上依然存在，人为造成市场分割。

深入整治"内卷式"竞争，关键在于统筹好有为政府与有效市场的关系。

第一，要着力矫正市场失灵，规范竞争秩序，使全国统一大市场成为各类经营主体公平竞争的大舞台。加强重点行业和领域监管执法，加大打假治劣力度，严厉打击恶意比价、虚假宣传、低价倾销等市场乱象。

第二，深入推进全国统一大市场建设，打破地区"小循环"。应进一步抓好《公平竞争审查条例》贯彻实施，着力破除各种形式的地方保护和市场分割，规范政府不当干预市场行为，拆除市场壁垒，为释放内需潜力奠定坚实基础。

第三，行业协会积极行动，广大企业自律担当。作为经济促进组织，行业协会是沟通政府、市场、企业和社会的关键一方，应在深入整治"内卷式"竞争中发挥更大的作用，规范企业行为，促进行业健康可持续发展。整治"内卷式"竞争，企业也应该成为破局者，看清大势，奋勇向价值链中高端攀登。

深入整治"内卷式"竞争，并不是否定竞争，而是让竞争回归创新和价值创造本身。中国经济是一片大海。随着整治"内卷式"竞争推向深入，相信有更多企业能在这片大海中成长为时代的弄潮儿，中国经济这艘巨轮也能在高质量发展的航道上，乘风破浪，行稳致远。

## 6. 🧠 核心考点与思维模型 (Go Pattern)

### Parser 逻辑流 (Pipeline)

将文章看作 `RawData`，你需要编写一个 `Parser` 将其转换为标准的 `AnalysisStruct`。

```go
type AnalysisStruct struct {
    Phenomenon string      // 现象描述
    Causes     []Cause     // 原因分析
    Impact     Impact      // 影响分析
    Solutions  []Solution  // 解决对策
    Summary    string      // 总结升华
}

type Cause struct {
    Level    string // 层面：行业、政府、市场、企业等
    Content  string // 具体内容
    Evidence string // 材料依据
}

type Solution struct {
    Subject  string // 主体：政府、市场、行业协会、企业等
    Action   string // 具体措施
    Goal     string // 预期效果
}
```

**Input (段1)**: 现象描述 —— **内卷式竞争**（"价格战"、"贴身肉搏"、"杀敌一千、自损八百"）

**Process A (段2)**: 原因一 —— **行业层面**（"新旧动能转换"、"存量竞争"、"焦虑情绪传导"）

**Process B (段7)**: 原因二 —— **政府层面**（"盲目追风口"、"铺摊子"、"变相补贴"）

**Process C (段7)**: 原因三 —— **市场层面**（"保护主义壁垒"、"市场分割"）

**Process D (段5)**: 原因四 —— **企业层面**（"资源投向外观包装"、"非核心技术攻关"）

**Process E (段2-6)**: 影响分析 —— **多重危害**（"扭曲市场机制"、"损害消费者权益"、"压缩创新空间"、"阻碍大循环"）

**Process F (段8-10)**: 对策 —— **多方协同**（政府矫正市场失灵、市场建设统一大市场、行业协会规范行为、企业向价值链高端攀登）

**Output (段11)**: 总结升华 —— **回归创新价值**（"让竞争回归创新和价值创造本身"、"高质量发展"）

### 答题技巧 (Middleware)

#### 1. **结构规范中间件**

**标准格式**：
```
总起句（现象本质）→ 原因分析（3-4点）→ 影响分析（简要）→ 解决对策（4-5点）→ 总结升华
```

**关键要素**：
- ✅ 总起句：用一句话概括现象本质（如："反映了...问题"、"折射出...矛盾"）
- ✅ 原因分析：使用"一是...二是...三是...四是..."的规范表述
- ✅ 对策分析：使用"一是...二是...三是...四是...五是..."的规范表述
- ✅ 总结升华：呼应开头，提出愿景

#### 2. **关键词抓取中间件**

**原因关键词**：
- "新旧动能转换"、"存量竞争"、"焦虑情绪传导"、"低价、低质、低水平竞争"
- "盲目追风口"、"铺摊子"、"重复建设"、"落后产能"、"变相补贴"、"资源错配"
- "保护主义壁垒"、"市场分割"、"人为造成"
- "资源投向外观包装"、"非核心技术攻关"、"削弱创新力"

**对策关键词**：
- "矫正市场失灵"、"规范竞争秩序"、"监管执法"、"打假治劣"、"严厉打击恶意比价、虚假宣传、低价倾销"
- "全国统一大市场建设"、"公平竞争审查条例"、"破除地方保护"、"拆除市场壁垒"
- "行业协会积极行动"、"规范企业行为"、"促进行业健康可持续发展"
- "企业成为破局者"、"向价值链中高端攀登"、"技术创新"、"产业升级"、"练好内功"

#### 3. **语言规范中间件**

**避免口语化**：
- ❌ "打价格战" → ✅ "陷入'内卷式'竞争泥潭"
- ❌ "互相压价" → ✅ "采取低价、低质、低水平竞争策略"
- ❌ "政府乱搞" → ✅ "地方政府盲目追风口、铺摊子"
- ❌ "企业不创新" → ✅ "企业缺乏创新动力，资源投向非核心环节"

**使用规范连接词**：
- 原因：首先、其次、再次、最后 / 一是、二是、三是、四是
- 对策：一是、二是、三是、四是、五是
- 总结：唯有...才能... / 只有...才能... / 让...回归...

#### 4. **逻辑衔接中间件**

**过渡句示例**：
- 原因与影响之间："这一现象导致了..." / "严重扭曲市场机制，损害..."
- 影响与对策之间："要解决这一问题，需要..." / "深入整治'内卷式'竞争，关键在于..."
- 对策之间："与此同时..." / "此外..." / "更重要的是..."

#### 5. **字数控制中间件**

**分配建议**（500字为例）：
- 总起句：40-50字
- 原因分析：160-180字（每点40-45字）
- 影响分析：40-50字
- 解决对策：200-220字（每点40-45字）
- 总结升华：40-50字

### 备考建议 (Best Practices)

1. **养成规范习惯**：
   - 始终使用"一是...二是..."的规范表述
   - 每个要点前加序号和"是"字
   - 保持段落层次清晰

2. **强化材料意识**：
   - 原因和对策都要从材料中提取，不能主观臆断
   - 注意材料中的关键词和关键句
   - 确保每个要点都有材料依据

3. **提升语言水平**：
   - 多积累规范表达（如"折射"、"凸显"、"破局"、"夯实"等）
   - 避免口语化表达
   - 注意句式变化，避免单一

4. **完善逻辑结构**：
   - 确保原因与对策一一对应
   - 注意各要点之间的逻辑关系
   - 结尾要有总结升华

5. **控制答题时间**：
   - 审题：2-3分钟
   - 阅读材料：5-8分钟
   - 组织答案：5-7分钟
   - 书写：10-15分钟
