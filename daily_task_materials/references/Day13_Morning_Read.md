## 🎯 Part 1: Day-12 属性规律识别技巧复习

### 1.1 属性规律的识别顺序（核心口诀）

**【三步识别法】**

> "**先看对称性，再看曲直性，最后看封闭性。**"

**【详细说明】**

1. **第一步：先看对称性**（最容易识别，优先判断）
   - 看图形是否能沿某条直线对折后完全重合 → **轴对称**
   - 看图形旋转180°后是否能与原图形重合 → **中心对称**
   - 数对称轴的数量和方向

2. **第二步：再看曲直性**（如果对称性不明显）
   - 看图形边缘是直线还是曲线
   - 区分"全直线"、"全曲线"、"曲直混合"
   - 注意：有些图形看似曲线，实际是折线（如锯齿形）

3. **第三步：最后看封闭性**（如果前两者都不明显）
   - 看图形是否有明显的开口
   - 区分"封闭图形"和"开放图形"
   - 注意：有些图形看似封闭，实际有微小开口

**【朗读练习】（3遍）**

> "属性规律的识别顺序：**先看对称性，再看曲直性，最后看封闭性**。  
> 对称性最容易识别，优先判断；如果对称性不明显，再看曲直性；如果前两者都不明显，最后看封闭性。"

---

### 1.2 对称性判断技巧

**【核心要点】**

- **快速识别**: 看图形是否有明显的对称轴
- **数对称轴**: 注意对称轴的数量（1条、2条、3条、4条、无数条）和方向（水平、垂直、斜向）
- **中心对称**: 看图形是否像"Z"、"S"、"平行四边形"等中心对称图形

**【常见图形识别】**

| 图形特征 | 对称类型 | 识别技巧 |
|---------|---------|---------|
| 能沿直线对折重合 | 轴对称 | 找对称轴 |
| 旋转180°后重合 | 中心对称 | 看是否像Z、S、平行四边形 |
| 圆 | 轴对称（无数条） | 任意直径都是对称轴 |
| 正方形 | 轴对称（4条） | 两条对角线+两条中位线 |

**【朗读练习】（3遍）**

> "对称性判断：**快速识别看对称轴，数对称轴看数量和方向，中心对称看Z、S、平行四边形**。  
> 轴对称是关于直线对称，中心对称是关于点对称。注意区分对称轴的数量和方向。"

---

### 1.3 曲直性判断技巧

**【核心要点】**

- **快速识别**: 看图形边缘是直线还是曲线
- **注意细节**: 有些图形看似曲线，实际是折线（如锯齿形）
- **混合图形**: 注意区分"曲直混合"和"全直线/全曲线"

**【常见类型】**

- **全直线**: 图形全部由直线段组成（如三角形、矩形、多边形）
- **全曲线**: 图形全部由曲线组成（如圆、椭圆、弧线）
- **曲直混合**: 图形既有直线又有曲线（如半圆+直线）

**【朗读练习】（3遍）**

> "曲直性判断：**快速识别看边缘，注意细节看折线，混合图形要区分**。  
> 全直线是全部直线段，全曲线是全部曲线，曲直混合是既有直线又有曲线。注意区分折线和曲线。"

---

### 1.4 封闭性判断技巧

**【核心要点】**

- **快速识别**: 看图形是否有明显的开口
- **注意细节**: 有些图形看似封闭，实际有微小开口
- **数量变化**: 注意封闭图形和开放图形的数量关系

**【常见类型】**

- **封闭图形**: 图形完全封闭，没有开口（如圆、矩形、三角形）
- **开放图形**: 图形有开口，不封闭（如U形、C形、L形）

**【朗读练习】（3遍）**

> "封闭性判断：**快速识别看开口，注意细节看微小开口，数量变化看关系**。  
> 封闭图形是完全封闭没有开口，开放图形是有开口不封闭。注意区分完全封闭和微小开口。"

---

### 1.5 属性规律的组合考查

**【常见组合】**

- **对称性 + 曲直性**: 如"轴对称 + 全曲线"
- **对称性 + 封闭性**: 如"中心对称 + 封闭图形"
- **曲直性 + 封闭性**: 如"全直线 + 开放图形"

**【避坑提醒】**

- 组合考查时，不要只关注一个属性而忽略另一个属性
- 注意区分"属性规律"和"样式规律"、"位置规律"

**【朗读练习】（2遍）**

> "属性规律可以组合考查：对称性+曲直性、对称性+封闭性、曲直性+封闭性。  
> 组合考查时，不要只关注一个属性而忽略另一个属性。"

## 🔍 Part 2: 综应"问题词"快速识别

### 2.1 直接问题词（核心词汇）

**【记忆口诀】**

> "**问题不足缺失缺，滞后薄弱困难多。**  
> **障碍矛盾冲突现，亟待迫切刻不容。**"

**【核心词汇】**

| 问题词 | 示例语境 |
|--------|---------|
| **问题** | 材料中反映的**问题**突出表现在... |
| **不足** | 当前工作存在明显**不足**... |
| **缺失** | 制度保障**缺失**，导致... |
| **缺乏** | **缺乏**有效的监管机制... |
| **滞后** | 管理手段**滞后**，无法适应... |
| **薄弱** | 基层力量**薄弱**，难以... |
| **困难** | 面临诸多**困难**和挑战... |
| **障碍** | 存在体制机制**障碍**... |
| **矛盾** | 供需**矛盾**日益突出... |
| **冲突** | 政策之间存在**冲突**... |
| **亟待** | 问题**亟待**解决... |
| **迫切** | **迫切**需要建立... |
| **刻不容缓** | 整改**刻不容缓**... |

**【朗读练习】（3遍）**

> "**问题**、**不足**、**缺失**、**缺乏**、**滞后**、**薄弱**、**困难**、**障碍**、**矛盾**、**冲突**。
