## 1. 核心概念 (Core Concepts)

### 1.1 假言命题 (Hypothetical Proposition)

假言命题是逻辑判断的核心，表示"条件关系"。

#### 1.1.1 如果A则B (If A then B)
*   **逻辑形式**: A → B
*   **含义**: A是B的充分条件，B是A的必要条件
*   **推理规则**:
    *   **肯前必肯后**: 如果A成立，则B一定成立 ✓
    *   **否后必否前**: 如果B不成立，则A一定不成立 ✓
    *   **否前不确定**: 如果A不成立，B可能成立也可能不成立 ✗
    *   **肯后不确定**: 如果B成立，A可能成立也可能不成立 ✗

**示例**:
- 如果下雨，则地面湿
- 肯前必肯后：下雨 → 地面湿 ✓
- 否后必否前：地面不湿 → 没下雨 ✓
- 否前不确定：没下雨 → 地面可能湿（洒水）也可能不湿 ✗
- 肯后不确定：地面湿 → 可能下雨也可能没下雨（洒水）✗

#### 1.1.2 只有A才B (Only if A then B)
*   **逻辑形式**: B → A（等价于：如果B则A）
*   **含义**: A是B的必要条件，B是A的充分条件
*   **推理规则**:
    *   **肯后必肯前**: 如果B成立，则A一定成立 ✓
    *   **否前必否后**: 如果A不成立，则B一定不成立 ✓
    *   **否后不确定**: 如果B不成立，A可能成立也可能不成立 ✗
    *   **肯前不确定**: 如果A成立，B可能成立也可能不成立 ✗

**示例**:
- 只有努力学习，才能考上公务员
- 肯后必肯前：考上公务员 → 努力学习 ✓
- 否前必否后：不努力学习 → 考不上公务员 ✓
- 否后不确定：没考上公务员 → 可能努力也可能不努力 ✗
- 肯前不确定：努力学习 → 可能考上也可能考不上 ✗

#### 1.1.3 除非A否则B (Unless A, otherwise B)
*   **逻辑形式**: 非A → B（等价于：如果非A则B）
*   **等价转换**: "除非A否则B" = "如果非A则B" = "只有A才非B"
*   **推理规则**: 按照"如果非A则B"的规则推理

**示例**:
- 除非下雨，否则去公园
- 等价于：如果不下雨，则去公园
- 等价于：只有下雨，才不去公园

### 1.2 联言命题 (Conjunctive Proposition)

*   **逻辑形式**: A 且 B
*   **含义**: A和B同时成立
*   **推理规则**:
    *   如果"A且B"为真，则A为真，B为真 ✓
    *   如果A为假或B为假，则"A且B"为假 ✓

### 1.3 选言命题 (Disjunctive Proposition)

#### 1.3.1 相容选言 (Inclusive Disjunction)
*   **逻辑形式**: A 或 B
*   **含义**: A和B至少有一个成立（可以同时成立）
*   **推理规则**:
    *   如果"A或B"为真，且A为假，则B为真 ✓
    *   如果"A或B"为真，且B为假，则A为真 ✓
    *   如果A为真，则"A或B"为真 ✓
    *   如果B为真，则"A或B"为真 ✓

#### 1.3.2 不相容选言 (Exclusive Disjunction)
*   **逻辑形式**: A 要么 B
*   **含义**: A和B有且仅有一个成立（不能同时成立）
*   **推理规则**:
    *   如果"A要么B"为真，且A为真，则B为假 ✓
    *   如果"A要么B"为真，且B为真，则A为假 ✓
    *   如果A为真且B为真，则"A要么B"为假 ✓
    *   如果A为假且B为假，则"A要么B"为假 ✓

## 2. 技巧与公式 (Techniques & Formulas)

### 2.1 假言命题推理口诀

```
肯前必肯后，否后必否前
否前不确定，肯后不确定
```

**记忆技巧**: 只有"肯前"和"否后"是确定的，其他都不确定。

### 2.2 逻辑等价转换

*   **如果A则B** = **非A或B** = **只有非B才非A**
*   **只有A才B** = **如果B则A** = **非A或B**（注意：这里A和B的位置）
*   **除非A否则B** = **如果非A则B** = **只有A才非B**

### 2.3 连锁推理 (Chain Reasoning)

如果A → B，B → C，则A → C（传递性）

**示例**:
- 如果努力学习 → 成绩好
- 如果成绩好 → 考上大学
- 则：如果努力学习 → 考上大学

### 2.4 解题步骤

1.  **识别题型**: 判断是假言、联言还是选言
2.  **转换形式**: 将自然语言转换为逻辑符号
3.  **应用规则**: 使用推理规则进行推导
4.  **验证选项**: 逐一验证选项是否符合推理规则

## 3. 避坑指南 (Pitfalls)

### 3.1 常见错误

*   **错误1**: 混淆"如果A则B"和"只有A才B"
    *   "如果A则B": A是充分条件
    *   "只有A才B": A是必要条件
*   **错误2**: 误用"否前"和"肯后"规则
    *   记住：只有"肯前"和"否后"是确定的
*   **错误3**: 混淆"或"和"要么"
    *   "或": 可以同时成立
    *   "要么": 不能同时成立
*   **错误4**: 忽略"除非...否则..."的转换
    *   必须转换为"如果非A则B"再推理

### 3.2 易错点

*   **自然语言陷阱**: 
    *   "只要A就B" = "如果A则B"
    *   "A是B的前提" = "只有A才B"
    *   "A是B的基础" = "只有A才B"
*   **双重否定**: 
    *   "不是不A" = "A"
    *   "没有不A" = "A"
*   **时间顺序**: 
    *   逻辑关系不等于时间顺序
    *   例如："先A后B"不等于"A → B"
