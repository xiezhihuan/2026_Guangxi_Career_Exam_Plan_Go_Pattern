# Day 3 重点笔记：数量关系 - 工程问题

> **返回任务卡**: [2025-12-07_Sunday.md](../../daily_tasks/2025-12-07_Sunday.md)

## 1. 核心概念 (Core Concepts)
*   **工程问题**: 研究工作量、工作效率、工作时间三者关系的应用题。
*   **核心公式**: 
    *   $$工作总量 = 工作效率 \times 工作时间$$
    *   $$工作效率 = \frac{工作总量}{工作时间}$$
    *   $$工作时间 = \frac{工作总量}{工作效率}$$

## 2. 技巧与公式 (Techniques & Formulas)

### 2.1 基础题型：单一工程
*   **特征**: 只有一项工程，一个人或一组人完成。
*   **解题步骤**:
    1.  设工作总量为"1"（或最小公倍数）。
    2.  根据条件求出工作效率。
    3.  根据问题求解。

### 2.2 常见变型：合作工程
*   **特征**: 多人合作完成一项工程。
*   **公式**: 
    *   若甲单独完成需 $a$ 天，乙单独完成需 $b$ 天，则合作效率 = $\frac{1}{a} + \frac{1}{b}$。
    *   合作时间 = $\frac{1}{\frac{1}{a} + \frac{1}{b}} = \frac{ab}{a+b}$。

### 2.3 进阶变型：交替工程
*   **特征**: 多人轮流工作。
*   **解题思路**: 先算一个周期的工作量，再看需要几个周期。

## 3. 避坑指南 (Pitfalls)
*   **单位统一**: 注意"天"、"小时"、"分钟"的单位换算。
*   **效率理解**: "效率提高20%"意味着新效率 = 原效率 × 1.2，而不是原效率 + 0.2。
*   **时间陷阱**: 问"提前几天"时，要用原时间减去实际时间。
