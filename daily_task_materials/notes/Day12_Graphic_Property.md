# Day 12 重点笔记：图形推理 - 属性规律

> **返回任务卡**: [2025-12-16_Tuesday.md](../../daily_tasks/2025-12-16_Tuesday.md)

## 1. 核心概念 (Core Concepts)

### 1.1 对称性 (Symmetry)
**定义**: 图形关于某条直线或某个点对称。

**类型**:
- **轴对称**: 图形关于某条直线对称
    - **识别技巧**: 看图形是否能沿某条直线对折后完全重合
    - **对称轴数量**: 1条、2条、3条、4条、无数条（圆）
    - **对称轴方向**: 水平、垂直、斜向（45°、135°）
- **中心对称**: 图形关于某个点对称（旋转180°后重合）
    - **识别技巧**: 看图形旋转180°后是否能与原图形重合
    - **常见图形**: 平行四边形、Z字形、S字形

**考查方式**:
- 所有图形都是轴对称
- 所有图形都是中心对称
- 对称轴数量递增/递减
- 对称轴方向变化

### 1.2 曲直性 (Curved vs Straight)
**定义**: 图形由曲线、直线或两者混合组成。

**类型**:
- **全直线**: 图形全部由直线段组成
- **全曲线**: 图形全部由曲线组成
- **曲直混合**: 图形既有直线又有曲线

**考查方式**:
- 所有图形都是全直线
- 所有图形都是全曲线
- 曲直性交替出现（直线→曲线→直线...）
- 曲直混合的数量变化

### 1.3 封闭性 (Open vs Closed)
**定义**: 图形是否封闭（是否有开口）。

**类型**:
- **封闭图形**: 图形完全封闭，没有开口
- **开放图形**: 图形有开口，不封闭

**考查方式**:
- 所有图形都是封闭的
- 所有图形都是开放的
- 封闭性交替出现（封闭→开放→封闭...）
- 封闭图形的数量变化

## 2. 技巧与公式 (Techniques & Formulas)

### 2.1 属性规律的识别顺序
1. **先看对称性**: 最容易识别，优先判断
2. **再看曲直性**: 如果对称性不明显，看曲直性
3. **最后看封闭性**: 如果前两者都不明显，看封闭性

### 2.2 对称性判断技巧
- **快速识别**: 看图形是否有明显的对称轴
- **数对称轴**: 注意对称轴的数量和方向
- **中心对称**: 看图形是否像"Z"、"S"、"平行四边形"等中心对称图形

### 2.3 曲直性判断技巧
- **快速识别**: 看图形边缘是直线还是曲线
- **注意细节**: 有些图形看似曲线，实际是折线（如锯齿形）
- **混合图形**: 注意区分"曲直混合"和"全直线/全曲线"

### 2.4 封闭性判断技巧
- **快速识别**: 看图形是否有明显的开口
- **注意细节**: 有些图形看似封闭，实际有微小开口
- **数量变化**: 注意封闭图形和开放图形的数量关系

### 2.5 属性规律的组合考查
- **对称性 + 曲直性**: 如"轴对称 + 全曲线"
- **对称性 + 封闭性**: 如"中心对称 + 封闭图形"
- **曲直性 + 封闭性**: 如"全直线 + 开放图形"

## 3. 避坑指南 (Pitfalls)

*   **易错点1**: 混淆"轴对称"和"中心对称"。轴对称是关于直线对称，中心对称是关于点对称。
*   **易错点2**: 对称轴数量判断错误。注意区分"1条对称轴"和"2条对称轴"。
*   **易错点3**: 对称轴方向判断错误。注意区分"水平对称轴"和"垂直对称轴"。
*   **易错点4**: 曲直性判断错误。注意区分"折线"（直线）和"曲线"。
*   **易错点5**: 封闭性判断错误。注意区分"完全封闭"和"有微小开口"。
*   **易错点6**: 属性规律组合时，只关注一个属性而忽略另一个属性。
*   **易错点7**: 误将"样式规律"或"位置规律"当作"属性规律"来解题。
