## 1. 核心概念 (Core Concepts)

### 1.1 现期倍数
**公式**: 倍数 = A / B
- A: 现期量（被比较的量）
- B: 现期量（比较基准）

**速算技巧**:
- 直接除法，保留2-3位有效数字
- 利用截位直除法快速计算
- 注意区分"是几倍"和"多几倍"：
  - "是几倍" = A / B
  - "多几倍" = (A - B) / B = A / B - 1

### 1.2 基期倍数
**公式**: 基期倍数 = (A/B) × [(1+b)/(1+a)]
- A: 现期量（被比较的量）
- B: 现期量（比较基准）
- a: A的增长率
- b: B的增长率

**速算技巧**:
- 先算现期倍数 A/B
- 再算 (1+b)/(1+a)，通常接近1，可估算
- 若 a < b，则基期倍数 > 现期倍数
- 若 a > b，则基期倍数 < 现期倍数

### 1.3 倍数与增长率的关系
**关系式**: 
- 若 A 是 B 的 n 倍，且 A 的增长率 = a，B 的增长率 = b
- 则倍数增长率 = (a - b) / (1 + b)

**判断技巧**:
- 若 a > b，则倍数上升
- 若 a < b，则倍数下降
- 若 a = b，则倍数不变

## 2. 技巧与公式 (Techniques & Formulas)

### 2.1 倍数比较
- **方法**: 比较两个量的增长率
- **口诀**: "被比较量快，倍数升；被比较量慢，倍数降"

### 2.2 倍数计算
- **截位直除**: 保留2-3位有效数字
- **首数法**: 只看第一位数字，快速判断大小
- **特征数字法**: 利用常见分数（1/2, 1/3, 1/4等）快速估算

### 2.3 倍数与增长量的关系
- 若问"A比B多多少"，注意区分：
  - "多几倍" = 倍数 - 1
  - "多多少量" = A - B（不是倍数关系）

## 3. 避坑指南 (Pitfalls)

*   **易错点1**: 混淆"是几倍"与"多几倍"。"是几倍"直接用除法，"多几倍"需要减1。
*   **易错点2**: 基期倍数计算时，忘记考虑增长率的方向（正负）。
*   **易错点3**: 倍数增长率公式中，分母是 (1+b) 而不是 (1+a)，容易记错。
*   **易错点4**: 单位不一致导致计算错误（如"亿元"与"万元"的换算）。
*   **易错点5**: 混淆"倍数"与"增长率"。倍数是两个量的比值，增长率是变化幅度。
*   **易错点6**: 在比较倍数大小时，误用现期倍数代替基期倍数进行比较。
