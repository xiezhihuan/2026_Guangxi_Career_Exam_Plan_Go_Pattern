# 错题管理指南

> **返回任务卡**: [2025-12-05_Friday.md](../../daily_tasks/2025-12-05_Friday.md)

请将每日做错的题目截图（或拍照），按日期和模块分类存放在此目录。

## 命名规范
`YYYYMMDD_Module_QuestionID.png`

**示例**:
*   `20251205_DataAnalysis_01.png` (12月5日资料分析第1题)
*   `20251205_Graphic_03.png` (12月5日图形推理第3题)

## 复盘三问
保存图片后，请在图片文件名或单独的Markdown文件中备注：
1.  **为什么错？** (计算错误? 公式记反? 没看懂题?)
2.  **正确思路是什么？** (解题的“金钥匙”在哪?)
3.  **下次怎么避坑？** (设定一个检查点)
