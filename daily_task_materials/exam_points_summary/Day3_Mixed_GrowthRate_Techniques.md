# 核心考点总结：混合增长率解题技巧

> **说明**: 本文档涵盖资料分析中混合增长率的三大核心题型：混合增长率（十字交叉法）、间隔增长率、年均增长率。

---

## 1. 混合增长率（十字交叉法）

### 1.1 核心概念
*   **定义**: 整体由两部分组成，已知两部分的增长率和现期量（或基期量），求整体的增长率。
*   **适用场景**: 
    *   整体 = 部分A + 部分B
    *   已知：部分A的增长率 $r_A$、部分B的增长率 $r_B$
    *   已知：部分A的现期量 $A$、部分B的现期量 $B$（或基期量）

### 1.2 核心公式

#### 方法一：加权平均法（精确计算）
$$r_{整体} = \frac{A \times r_A + B \times r_B}{A + B}$$

**推导过程**:
*   整体现期量 = $A + B$
*   整体基期量 = $\frac{A}{1+r_A} + \frac{B}{1+r_B}$
*   整体增长率 = $\frac{(A+B) - (\frac{A}{1+r_A} + \frac{B}{1+r_B})}{\frac{A}{1+r_A} + \frac{B}{1+r_B}}$

**简化形式**（当 $r_A$、$r_B$ 较小时）:
$$r_{整体} \approx \frac{A \times r_A + B \times r_B}{A + B}$$

#### 方法二：十字交叉法（速算技巧）
**核心逻辑**: 混合增长率介于两部分增长率之间，且偏向现期量大的部分。

**十字交叉法步骤**:
1.  设整体增长率为 $r$，部分A增长率为 $r_A$，部分B增长率为 $r_B$（假设 $r_A > r_B$）
2.  列出十字交叉式：
    ```
        部分A:  r_A    |r - r_B|
        整体:    r
        部分B:  r_B    |r_A - r|
    ```
3.  比例关系：$\frac{|r - r_B|}{|r_A - r|} = \frac{A}{B}$（现期量之比）
4.  解方程求 $r$

**速算口诀**:
*   **整体增长率介于两部分增长率之间**
*   **偏向现期量大的部分**
*   **现期量之比 = 距离反比**

### 1.3 实战例题

**例题1**:
> 2023年某市GDP为5000亿元，其中第一产业500亿元（增长5%），第二产业3000亿元（增长8%），第三产业1500亿元（增长6%）。求2023年该市GDP的混合增长率。

**解析**:
*   方法一（加权平均）:
    $$r = \frac{500 \times 5\% + 3000 \times 8\% + 1500 \times 6\%}{5000}$$
    $$= \frac{25 + 240 + 90}{5000} = \frac{355}{5000} = 7.1\%$$

*   方法二（十字交叉，先算第一+第二产业）:
    *   第一产业：5%，现期量500
    *   第二产业：8%，现期量3000
    *   整体增长率介于5%和8%之间，偏向3000（第二产业）
    *   距离比 = $\frac{500}{3000} = \frac{1}{6}$
    *   设整体增长率为 $r$，则 $\frac{r-5\%}{8\%-r} = \frac{3000}{500} = 6$
    *   解得：$r = 7.14\% \approx 7.1\%$

**速算技巧**:
*   当现期量差距很大时，整体增长率接近现期量大的部分的增长率
*   例：若第二产业3000亿（8%），第三产业1500亿（6%），整体增长率更接近8%

## 2. 间隔增长率

### 2.1 核心概念
*   **定义**: 已知连续两年的增长率，求间隔一年的增长率。
*   **场景**: 已知2021年增长率 $r_1$，2022年增长率 $r_2$，求2022年相对于2020年的增长率 $R$。

### 2.2 核心公式

$$R = r_1 + r_2 + r_1 \times r_2$$

**推导过程**:
*   设2020年基期量为 $A$
*   2021年现期量 = $A(1+r_1)$
*   2022年现期量 = $A(1+r_1)(1+r_2) = A(1+r_1+r_2+r_1r_2)$
*   间隔增长率 = $\frac{A(1+r_1+r_2+r_1r_2) - A}{A} = r_1 + r_2 + r_1r_2$

### 2.3 速算技巧

**口诀**: "和+积"

**速算方法**:
1.  **当 $r_1$、$r_2$ 都小于10%时**:
    *   $r_1 \times r_2$ 很小，可近似忽略
    *   $R \approx r_1 + r_2$

2.  **当 $r_1$、$r_2$ 有一个较大（>10%）时**:
    *   先算 $r_1 + r_2$
    *   再算 $r_1 \times r_2$（可用估算）
    *   相加得结果

**常用估算**:
*   $10\% \times 10\% = 1\%$
*   $20\% \times 15\% = 3\%$
*   $25\% \times 20\% = 5\%$

### 2.4 实战例题

**例题2**:
> 2021年某产品销量增长15%，2022年增长20%。求2022年相对于2020年的间隔增长率。

**解析**:
$$R = 15\% + 20\% + 15\% \times 20\%$$
$$= 35\% + 3\% = 38\%$$

**速算**: $15\% + 20\% = 35\%$，$15\% \times 20\% = 3\%$，合计38%

## 3. 年均增长率

### 3.1 核心概念
*   **定义**: 已知基期量和现期量，求年均增长率。
*   **场景**: 已知2020年基期量 $A$，2023年现期量 $B$，求年均增长率 $r$。

### 3.2 核心公式

#### 精确公式（开方）
$$B = A(1+r)^n$$
$$r = \sqrt[n]{\frac{B}{A}} - 1$$

其中 $n$ 为间隔年数（注意：2020到2023是3年，$n=3$）

#### 近似公式（当 $r$ 较小时）
$$r \approx \frac{\frac{B}{A} - 1}{n} = \frac{总增长率}{年数}$$

**注意**: 此公式仅当 $r$ 较小时适用，通常 $r < 10\%$ 时误差较小。

### 3.3 速算技巧

#### 技巧一：选项差距大时用估算
*   计算 $\frac{B}{A}$，判断大概倍数
*   例：$\frac{B}{A} = 1.5$，$n=3$，则 $r \approx \frac{0.5}{3} = 16.7\%$
*   实际值略小于此（因为 $(1+r)^3$ 的展开）

#### 技巧二：常用开方值速查
| 倍数 $\frac{B}{A}$ | n=2 | n=3 | n=4 | n=5 |
| :--- | :--- | :--- | :--- | :--- |
| 1.21 | 10% | - | - | - |
| 1.44 | 20% | - | - | - |
| 1.5 | - | ~14.5% | - | - |
| 1.69 | 30% | ~19% | - | - |
| 2.0 | 41.4% | ~26% | ~19% | ~15% |

#### 技巧三：代入验证法
*   当选项给出具体数值时，代入 $(1+r)^n$ 验证
*   例：选项有15%、20%、25%，分别计算 $(1.15)^3$、$(1.20)^3$、$(1.25)^3$

### 3.4 实战例题

**例题3**:
> 2020年某市GDP为4000亿元，2023年GDP为6000亿元。求2020-2023年的年均增长率。

**解析**:
*   $\frac{B}{A} = \frac{6000}{4000} = 1.5$
*   $n = 3$（2020到2023是3年）
*   精确计算：$r = \sqrt[3]{1.5} - 1 \approx 14.5\%$
*   近似计算：$r \approx \frac{0.5}{3} = 16.7\%$（偏大，需下调）

**答案**: 约14.5%

## 4. 综合解题策略

### 4.1 题型识别
1.  **混合增长率**: 题目中出现"整体"、"合计"、"总计"，且给出各部分增长率
2.  **间隔增长率**: 题目明确问"间隔一年"或"相对于前两年"
3.  **年均增长率**: 题目问"年均增长"、"平均每年增长"

### 4.2 计算优先级
1.  **混合增长率**: 优先用十字交叉法（现期量差距大时直接估算）
2.  **间隔增长率**: 先算和，再算积，相加
3.  **年均增长率**: 选项差距大用估算，差距小用开方或代入验证

### 4.3 易错点
*   **混合增长率**: 
    *   注意现期量还是基期量（通常用现期量）
    *   整体增长率一定介于两部分增长率之间
*   **间隔增长率**: 
    *   注意是"间隔一年"还是"连续两年"（间隔一年用公式，连续两年直接给）
    *   乘积项不要漏掉
*   **年均增长率**: 
    *   **年数计算**: 2020到2023是3年，不是4年！
    *   近似公式仅适用于增长率较小的情况
    *   开方计算可用计算器或估算

---

## 5. 实战速算口诀总结

### 混合增长率（十字交叉）
*   **整体介于部分之间，偏向现期量大的**
*   **现期量比 = 距离反比**

### 间隔增长率
*   **和+积，小可忽略**

### 年均增长率
*   **年数要算对，开方或估算**
*   **选项差距大用估算，差距小用验证**
