# 项目概述与核心原则

## 项目性质
这是一个**2026年广西事业单位考试备考项目**，专为在职考生设计，采用Go语言设计模式作为组织隐喻。

## 核心原则

### 1. Go语言设计模式隐喻
- 使用Go语言的设计模式、并发模型、系统架构等概念来组织备考策略
- 例如：Goroutine调度、Pipeline处理、Circuit Breaker熔断机制等
- 所有文档和策略都应体现这种技术思维

### 2. 在职备考优化
- **核心痛点**：时间碎片化、下班后精力不足、计划易中断
- **解决方案**：时间分片（Time Slicing）、上下文切换（Context Switching）、熔断降级

### 3. 科学记忆规律
- 遵循**艾宾浩斯遗忘曲线**：Day-1、Day-3、Day-7滚动复习
- 应用**认知负荷理论**：控制新知识容量，学练结合

### 4. 结构化文档管理
- 所有学习材料按日期和主题分类存储
- 使用Markdown格式，支持双向链接
- 保持文档的可追溯性和可维护性

## 目标用户画像
- **专业**：电子信息工程
- **学历**：本科
- **地点**：南宁
- **工作状态**：在职（09:00-18:00）
- **特点**：晚间电量低，需要利用碎片时间

## 考试目标
- **考试时间**：2026年3月底（预计3月30日）
- **考试科目**：
  - 职业能力倾向测验（90分钟，100题）
  - 综合应用能力A类（120分钟，3-4题）
- **报考类别**：A类（综合管理）或C类（自然科学专技）
