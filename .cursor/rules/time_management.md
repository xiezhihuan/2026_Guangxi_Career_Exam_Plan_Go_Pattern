# 时间管理与任务调度规则

## Goroutine调度模型（G1-G4）

项目采用Go语言的Goroutine并发模型来组织每日学习时间，将一天分为4个并发协程：

### G1: Early Morning Worker（晨间协程）
- **时间**：06:30 - 07:30（60分钟）
- **优先级**：High
- **上下文**：Background（大脑最清醒，无工作打扰）
- **任务类型**：CPU密集型任务
- **典型任务**：
  - 做2篇言语理解 / 1组资料分析
  - 背诵综应的核心考点
  - 学习新知识点（理论+公式）
- **原则**：早上时间完全可控，是学习的黄金时间

### G1.5: Morning Buffer（晨间缓冲）
- **时间**：07:30 - 08:30（60分钟）
- **优先级**：Medium
- **上下文**：Life Support
- **任务序列**：
  - **07:30-08:00**：营养早餐 & 洗漱整理（Life Support）
  - **08:00-08:30**：晨读/预热（Light IO）
    - 成语积累
    - 综应金句朗读
    - 错题本快速浏览
- **原则**：低强度记忆工作，为G2/G3准备

### G2: Commute Microservice（通勤微服务）
- **时间**：08:30 - 09:00（30分钟）
- **优先级**：Low
- **上下文**：Mobile（公交/地铁/开车，环境嘈杂）
- **任务类型**：IO等待型任务
- **典型任务**：
  - **听觉**：听时政热点磨耳朵（半月谈、人民日报）
  - **视觉**：手机刷常识题（刷完即忘，混眼熟）
- **原则**：不求甚解，只求混眼熟

### G3: Lunch Break Worker（午休协程）
- **时间**：12:00 - 13:30（90分钟）
- **优先级**：Medium
- **上下文**：Office（需要快速切换状态）
- **任务序列**：
  - **12:00-12:20（Motion）**：简单运动（拉伸/散步），激活身体
  - **12:20-12:40（Study）**：刷15道图形推理 / 错题浏览（利用运动后的兴奋期）
  - **12:40-13:00（Eat）**：快速进食，补充能量
  - **13:00-13:30（Sleep）**：强制午睡（即使睡不着也要闭目养神）
- **原则**：保证下午和G4的电量

### G4: Night Batch Job（晚间批处理）
- **时间**：20:00 - 23:00（180分钟）
- **优先级**：Critical
- **上下文**：Deep Work（容易疲惫，需对抗惰性）
- **任务类型**：系统性学习
- **时间分配**：
  - **20:00-21:00**：听网课/学新知识点
  - **21:00-22:00**：专项刷题训练
  - **22:00-22:30**：综应写作练习（哪怕只写提纲）
  - **22:30-23:00**：洗澡/放松/早睡（保证G1重启成功）
- **原则**：系统性学习，但必须23:00前睡觉

## 时间分配策略

### 职业能力倾向测验（90分钟）
| 模块 | 题量 | 建议时间 | 策略 |
|------|------|----------|------|
| 资料分析 | 15题 | 20分钟 | 必须拿满分 |
| 言语理解 | 25题 | 32分钟 | 核心拿分点 |
| 判断推理 | 30题 | 30分钟 | 图形推理快，逻辑仔细 |
| 常识判断 | 20题 | 8分钟 | 秒选，不纠结 |
| 数量关系 | 10题 | 10分钟 | 只做简单题 |
| 涂卡+检查 | - | 5分钟 | 预留缓冲 |

### 做题顺序（推荐）
1. 资料分析（最有把握）
2. 言语理解（趁脑子清醒）
3. 判断推理（图形快，逻辑仔细）
4. 常识判断（秒选）
5. 数量关系（最后做，只做简单题）

### 综合应用能力A类（120分钟）
- **审题+读材料**：20分钟（不能省）
- **第1题（案例分析）**：25分钟
- **第2题（案例分析）**：25分钟
- **第3题（应急处理）**：25分钟
- **第4题（公文写作）**：30分钟
- **检查**：5分钟

## 异常处理与熔断机制

### Circuit Breaker（熔断器）
当工作压力超过阈值时，触发熔断：
- **Keep Alive模式**：只保留最低限度的维持（刷10道题）
- **Context Cancellation**：如果必须加班到很晚，直接取消当晚复习计划
- **原则**：不要熬夜补进度，否则会引发连锁崩溃

### Deadline控制
- 每个模块到时间必须停笔，进入下一模块
- 如果某道题超过2分钟还没思路，直接跳过，标记后回头再看
