# 命令：/analyze_wrong_questions

本规则定义了如何对用户在错题本中记录的错题进行 **智能分析**，总结错误类型、知识盲点和改进建议，形成系统化的复习材料。

## 触发方式

用户输入：`/analyze_wrong_questions` 或提供错题本文件路径

## 核心逻辑：错题分析系统 (Wrong Questions Analysis System)

当用户完成练习并记录错题后，执行此命令，AI 将扮演 **错题分析师** 的角色，执行以下分析流程：

1.  **Data Extraction (数据提取)**
    *   **读取错题本**: 扫描错题本文件，提取所有错题信息（题目、答案、错误原因等）。
    *   **统计练习数据**: 提取总题数、正确数、错误数、正确率、用时等统计信息。

2.  **Pattern Analysis (模式分析)**
    *   **错误类型分类**: 根据错误原因，将错题分类（如：关键词理解偏差、选非题审题不清、规则理解错误、关键词遗漏等）。
    *   **知识盲点识别**: 识别用户的知识盲点（如：概念混淆、规则理解不清、审题习惯问题等）。
    *   **易错点总结**: 总结高频错误点和易混淆概念。

3.  **Output Generation (输出生成)**
    *   **错误类型统计**: 生成错误类型统计表，显示各类错误的题数和占比。
    *   **核心问题总结**: 提炼出3-5个核心问题，每个问题配以典型题目和改进建议。
    *   **知识盲点梳理**: 按学科或知识点分类，整理知识盲点。
    *   **复习计划**: 制定立即复习、短期复习、中期复习计划。
    *   **记忆口诀**: 为易错知识点生成记忆口诀。

## 执行步骤 (Execution Pipeline)

1.  **Read Wrong Questions File (读取错题本)**
    *   读取用户指定的错题本文件（通常在 `daily_task_materials/wrong_questions/` 目录下）。
    *   如果用户未指定文件，自动查找最新的错题本文件。

2.  **Extract Key Information (提取关键信息)**
    *   提取每道错题的以下信息：
        *   题目内容
        *   用户答案 vs 正确答案
        *   错误原因分析
        *   正确解析
        *   改进建议
        *   用时和正确率

3.  **Analyze Patterns (分析模式)**
    *   **错误类型分类**:
        *   关键词理解偏差
        *   选非题审题不清
        *   规则理解错误
        *   关键词遗漏
        *   概念混淆
        *   时间/空间要素遗漏
        *   双重定义抓取不完整
        *   其他
    *   **知识盲点识别**:
        *   按学科分类（如：法律、地理、政治、科技、社会学、心理学、管理学、经济学、数学等）
        *   按概念分类（如：行政强制 vs 行政处罚、初级群体 vs 次级群体等）

4.  **Generate Analysis Report (生成分析报告)**
    *   在错题本文件末尾追加分析报告，使用分割线 `---` 与原有内容隔开。

## 适用场景

*   **定义判断错题**: 分析定义判断练习中的错题，总结关键词抓取、选项匹配等问题。
*   **资料分析错题**: 分析资料分析练习中的错题，总结公式记忆、计算错误等问题。
*   **图形推理错题**: 分析图形推理练习中的错题，总结规律识别、特征提取等问题。
*   **言语理解错题**: 分析言语理解练习中的错题，总结中心句识别、语境分析等问题。
*   **常识判断错题**: 分析常识判断练习中的错题，总结知识盲点、概念混淆等问题。
*   **数量关系错题**: 分析数量关系练习中的错题，总结公式应用、解题思路等问题。

## 行为准则

*   **客观分析**: 基于错题本中的实际错误，不要主观臆断，确保分析准确。
*   **分类清晰**: 错误类型分类要准确，避免重复和遗漏。
*   **建议可执行**: 改进建议要具体可操作，不要泛泛而谈。
*   **口诀易记**: 记忆口诀要简洁明了，便于记忆。
*   **格式规范**: 确保生成的 Markdown 格式整洁，重点内容加粗，表格对齐。

## 错误类型分类标准

### 1. 关键词理解偏差
- **特征**: 对定义中的关键词理解不准确，导致选项匹配错误
- **典型场景**: 混淆相似概念、理解偏差、过度解读
- **改进方向**: 强化关键词理解，对比记忆易混淆概念

### 2. 选非题审题不清
- **特征**: 选非题时未明确标记"不涉及"、"不属于"等关键词
- **典型场景**: 选非题、反向思维题
- **改进方向**: 明确标记关键词，反向思考

### 3. 规则理解错误
- **特征**: 对题目规则或操作步骤理解有偏差
- **典型场景**: 逐步操作题、编码题、计算题
- **改进方向**: 严格按照步骤操作，注意循环规则

### 4. 关键词遗漏
- **特征**: 遗漏定义中的关键要素（如时间、空间、权限等）
- **典型场景**: 定义中有多个关键要素，只匹配了部分
- **改进方向**: 全面抓取关键词，注意限制性要素

### 5. 概念混淆
- **特征**: 混淆两个或多个相似概念
- **典型场景**: 行政强制 vs 行政处罚、初级群体 vs 次级群体
- **改进方向**: 对比记忆，找出区别点

### 6. 时间/空间要素遗漏
- **特征**: 忽略了定义中的时间、空间等限制性要素
- **典型场景**: "最近接触"、"同等权限"等限制性要素
- **改进方向**: 注意定义中的时间、空间、权限等限制性要素

### 7. 双重定义抓取不完整
- **特征**: 当定义有两个关键信息时，只匹配了其中一个
- **典型场景**: 定义中有"也指"、"或者"等并列关系
- **改进方向**: 确保选项同时符合定义的所有关键信息

### 8. 其他
- **特征**: 不属于以上类型的其他错误
- **典型场景**: 计算错误、粗心大意等
- **改进方向**: 根据具体情况分析

## 知识盲点分类标准

### 按学科分类
- **法律**: 行政强制、行政处罚、行政许可、民法典等
- **地理**: 气候类型、地形特征、区域划分等
- **政治**: 政治制度、国家体制、政党制度等
- **科技**: 卫星导航系统、大气污染、生活常识等
- **社会学**: 初级群体、次级群体、社会分层等
- **心理学**: 对比效应、首因效应、近因效应等
- **管理学**: 协议制决策、垂直管理、扁平化管理等
- **经济学**: 形象转移、公共产品、外部性等
- **数学**: N后置编码、排列组合、概率等

### 按概念分类
- **易混淆概念对**: 
  - 行政强制 vs 行政处罚
  - 初级群体 vs 次级群体
  - 经验假说 vs 理论假说
  - 探索性调研 vs 描述性调研
  - 等等

## 记忆口诀生成原则

1. **简洁明了**: 口诀要简短，便于记忆
2. **逻辑清晰**: 口诀要体现知识点的逻辑关系
3. **朗朗上口**: 口诀要押韵或对仗，便于朗读
4. **重点突出**: 口诀要突出关键要素和区别点

4. 生成分析报告并追加到错题本文件末尾
5. 提供具体的改进建议和复习计划

- 通过记忆口诀快速巩固知识点
